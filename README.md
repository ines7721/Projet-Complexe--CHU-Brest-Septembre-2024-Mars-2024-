# Projet-Complexe--CHU-Brest-Septembre-2024-Mars-2024-
Projet complexe de troisième année (IMT Atlantique) en partenariat avec le laboratoire LATIM au CHU de Brest. Traduction Python de la méthode ACP (Automated Cutting Plane) pour l'analyse morphologique des anévrismes intracrâniens.

Le travail étant soumis à des contraintes de propriété intellectuelle, seul le readme servant d'évaluation du projet peut être mis à disposition publique.
